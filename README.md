# Vigor

My personal neovim configuration. The `vim` branch has the vim8 configuration.

## Usage

```sh
git clone git://github.com/arnau/Vigor.git ~/.config/nvim
nvim --headless -u NONE -c 'lua require("bootstrap").bootstrap_paq()'
```
