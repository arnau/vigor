# Vigor

Vigor is Arnau Siches' Vim Playground.

## Usage

First, clone the repository.

    $ git clone git://github.com/arnau/Vigor.git ~/.vim
    $ cd ~/.vim
    $ ./vigor install

If it's already installed you can update it running

    $ cd ~/.vim
    $ ./vigor update
